## GitHub Pages Setup

### Quick Start

1. **Enable GitHub Pages**
   - Go to your repository settings
   - Navigate to "Pages" section
   - Under "Source", select `main` branch
   - Click "Save"
   - Your site will be published at: `https://yourusername.github.io/Data-viz/`

2. **Access Your Portfolio**
   - Wait 2-3 minutes for GitHub to build your site
   - Visit `https://nikhilkumarkorrapati.github.io/Data-viz/`
   - Your portfolio is now live!

### Customization

To personalize your portfolio:

1. **Update Contact Information** in `index.html`:
   ```html
   <!-- Search for and update these sections -->
   - Email address
   - LinkedIn URL
   - GitHub URL
   - Twitter/Social media links
   ```

2. **Add Your Own Projects**:
   - Place project files in the `projects/` folder
   - Update project cards in `index.html`
   - Follow the existing project structure

3. **Modify Colors & Styling**:
   - Edit CSS variables in `assets/css/style.css`
   - Customize the color scheme in the `:root` section
